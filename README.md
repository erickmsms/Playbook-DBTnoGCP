# DBT-no-GCP-Guia-Pratico-e-Manual-de-Producao
Este repositório reúne dois documentos complementares sobre o uso de DBT no GCP em ambientes de produção. Um guia executável, com passo a passo claro para implementar pipelines de transformação, e um manual técnico aprofundado, com princípios, padrões e boas práticas aplicadas em projetos reais de engenharia de dados.
